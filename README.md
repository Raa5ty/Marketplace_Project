# 🛒 Маркетплейс - Интернет-магазин на Django

Проект интернет-магазина, разработанный в рамках учебного проекта. Реализует отображение списка товаров и детальных страниц с использованием Django Class-Based Views, шаблонов с наследованием и статических файлов.

## 🚀 Функционал

- **Список товаров** - главная страница с каталогом всех товаров
- **Детальная страница товара** - полная информация о товаре
- **Админ-панель** - управление товарами (CRUD операции)
- **Загрузка изображений** - поддержка картинок для каждого товара
- **Адаптивные изображения**:
  - На странице списка - уменьшенные миниатюры (200px)
  - На детальной странице - увеличенные изображения (до 500px)
- **Обработка ошибок** - автоматический возврат 404 при несуществующем товаре
- **Фильтры шаблонов**:
  - Форматирование цены (2 знака после запятой)
  - Обрезка длинных описаний
  - Форматирование даты
  - Сохранение переносов строк в описании

## 🛠 Технологии

- **Backend**: Django 4.x / 5.x
- **Frontend**: HTML5, CSS3
- **База данных**: SQLite (по умолчанию)
- **Изображения**: Pillow
