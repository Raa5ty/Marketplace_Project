# 🛒 Маркетплейс - Интернет-магазин на Django

Проект интернет-магазина, разработанный в рамках учебного проекта. Реализует отображение списка товаров и детальных страниц с использованием Django Class-Based Views, шаблонов с наследованием и статических файлов.

Рекомендую ознакомится с пунктами: **"Как запустить проект**" и **"Особенности работы"**

## 🚀 Функционал

- Каталог товаров с категориями
- Детальная страница товара с отзывами (рейтинг 1-5)
- Корзина (для авторизованных и анонимных пользователей с переносом при входе)
- Система регистрации и авторизации (email как логин)
- Активация аккаунта по email (токен)
- Восстановление пароля (письмо с ссылкой)
- Личный кабинет (редактирование профиля, смена пароля, история заказов)
- Админ-панель для управления товарами, заказами, пользователями

## 🛠 Технологии

- **Backend**: Django 4.x / 5.x
- **Frontend**: HTML5, CSS3
- **База данных**: SQLite (по умолчанию)
- **Изображения**: Pillow

## 🚀 Как запустить проект

###### Клонирование репозитория

```
git clone <url-репозитория>
cd Marketplace_Project
```

###### Создание виртуального окружения

```
# Windows
python -m venv .venv
.venv\Scripts\activate

# macOS/Linux
python3 -m venv .venv
source .venv/bin/activate
```

###### Установка зависимостей

```
pip install django pillow
```

###### Применение миграций, создание суперпользователя, запуск сервера

```
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

* **Главная страница** : [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
* **Админ-панель** : [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)

## Особенности работы

#### Регистрация и активация

1. Заполните форму регистрации (email, пароль, телефон, адрес)
2. В **терминале** появится ссылка для активации (письма на реальную почту не отправляются)
3. Скопируйте ссылку и откройте в браузере для активации аккаунта

#### Восстановление пароля

1. На странице входа нажмите "Забыли пароль?"
2. Введите email, указанный при регистрации
3. В **терминале** появится ссылка для сброса пароля
4. Перейдите по ссылке и установите новый пароль

#### Корзина

* Анонимные пользователи: товары сохраняются в сессии браузера
* При входе в аккаунт: товары из сессионной корзины автоматически переносятся в аккаунт
* Корзина сохраняется после перезахода


## Примечание

* Письма с активацией и сбросом пароля выводятся в **консоль** (настроен `console.EmailBackend`)
* Для реальной отправки писем замените в `settings.py`:

```
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.yandex.ru'  # или другой SMTP
```
